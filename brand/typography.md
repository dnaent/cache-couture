# Caché — Typography

> Source of truth: CLAUDE.md §2.4. Keep this file in sync.

## Display / Wordmark
- **Family:** Didone serif (Bodoni, Didot, or a custom cut)
- **Treatment:** capitalised, classical proportions, high optical contrast
- **Used on:** `CACHÉ` wordmark, `COUTURE` lockup, hero campaign titles

## Subline / Tagline
- **Family:** tracked small-caps serif (same Didone family)
- **Treatment:** generously letter-spaced
- **Signature device:** the second token of `HIDDEN / APPROVAL` is rendered
  mirrored/upside-down. This must be preserved wherever the tagline appears.

## Body / Technical
- **Family:** neutral grotesk — Inter, Söhne, or Helvetica Now
- **Treatment:** light to regular weight, restrained leading
- **Used on:** spec sheets, drop pages, metadata, NFT certificates, sidecar JSON

## Headline structure
- Noun-led, often one or two kerned-out caps
- Examples: `ENCRYPTED LUXURY.` · `STEALTH PROTOCOL.` · `TETHERED.`

## Forbidden treatments
- Bold weights on the wordmark (always thin/light/regular for Didone display)
- Italics on body grotesk
- All-caps body copy
- Coloured type — type stays in the monochromatic palette (see `palette.json`)

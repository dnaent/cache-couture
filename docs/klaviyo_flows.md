> Paste-ready email/flow copy in Caché brand voice (CLAUDE.md §6: cold, minimal, technical; no hype,
> no exclamation marks, no emoji). Each email = Subject / Preview text / Body. Timing per email.
> Links use branded URLs (`hiddencache.co.uk/...`). Created 2026-07-01.

## Setup notes
- **Sender:** From name `Caché`, from email `contact@hiddencache.co.uk`.
- **No discount codes by default** — Caché is stealth-luxury; scarcity/"limited access" language outperforms discounting and protects the brand. Introduce codes only deliberately.
- **Phygital wording** stays "coming soon / your SKU reserves it" until the NFT mint is live (V1 ships without live minting).
- **Every footer** must carry: DNA Entertainment Ltd + registered address + [unsubscribe] + [Privacy Policy](https://hiddencache.co.uk/policies/privacy-policy) (legal requirement).
- **Timing** is a starting point — Klaviyo lets you A/B and adjust delays later.
